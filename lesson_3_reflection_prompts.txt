Quando gostaria de usar um repositório remoto em vez de manter o trabalho local?

	Resp. Sempre, pois além de trazer segurança como um backup, podemos continuar trabalhando mesmo quando não estamos em nosso computador e no compartilhamento com a comunidade e para trabalho em equipe.
  
Por que você prefere efetuar pull de alterações manualmente do que deixar o Git atualizar-se automaticamente com o seu repositório remoto?

	Resp. Pois um projeto que gostariamos de compartilhar somente depois de pronto não poderia ser enviado automaticamente ao github.

Descreva as diferenças entre bifurcações, clones e branches. Quando você usaria um e não os outros?

	Resp. Bifurcação é um clone de outro repositorio dentro do github, já clones são cópias de repositórios do github para um computador local e branchs são rótulos de commits dentro de qualquer um desses repositórios.

Qual é o benefício de ter uma cópia do último estado conhecido do remoto armazenado localmente?

	Resp. Permite-nos saber onde foi o último push e a analisar as diferenças entre a versão remota e a local. 

Como você colaboraria sem usar Git ou GitHub? O que seria mais fácil e mais difícil?

	Resp. Definiria uma parte ou funcionalidade especifica para trabalhar em coisas diferentes dos outros colaboradores. A parte difícil seria não poder verificar facilmente as alterações dos colaboradores e não ter ajuda em algum ponto que estou trabalhando sozinho.
